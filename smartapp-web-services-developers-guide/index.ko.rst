웹 서비스 SmartApps
======================

SmartApps는 그 자체로 URL이나 정의된 종점을 노출시키는 웹 서비스가 될 수 있습니다.

이를 통해 외부 어플리케이션이 SmartApp에 웹 API 호출을 하고 단말기에 대한 정보를 얻거나 제어할 수 있습니다.
